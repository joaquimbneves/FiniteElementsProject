# FiniteElementsProject
2026
Project developed for Mecanica Computacional (Computational Mechanics).


Development of Finite elements T3 (three node triangles) for a simple mesh, defined in MatLab and a complex mesh, defined in Ansys and imported to MatLab.
Used to analyse the behaviour of an ideal fluid around the geometries defined.
In the final report a comparison is made between the developed code and commercial software Ansys.
